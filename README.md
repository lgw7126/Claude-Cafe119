# ☕ 1초 레시피 & 비상 매뉴얼 (Cafe Quick-Cheat App)

🔗 **[Live Demo → https://lgw7126.github.io/Claude-Cafe119/](https://lgw7126.github.io/Claude-Cafe119/)**

카페·샌드위치 가게 등 신입 알바생을 위한 **오프라인 단일 파일** 커닝페이퍼 웹앱입니다.
서버·데이터베이스·외부 라이브러리 없이 `index.html` 파일 하나로 동작합니다.

## 주요 기능
- **초성 실시간 검색** — `ㅂㄴㄹ` → 바닐라라떼처럼 초성만 입력해도 즉시 검색
- **반응형 UI** — PC 가로(2열) / 모바일 세로(1열) 자동 적응, 큰 글씨로 가독성 확보
- **키워드 자동 강조** — `[재료 위치]`는 노란색, 전화번호는 빨간색으로 자동 표시
- **사장님 관리 모드** (PIN 잠금, 기본 `0000`)
  - 레시피·비상매뉴얼 **추가 / 수정 / 삭제**
  - 브라우저(localStorage) 자동 저장 + **JSON 백업 내보내기 / 불러오기**
  - 기본값으로 초기화
- **오프라인 PWA** — '홈 화면에 추가'로 인터넷 없이 앱처럼 사용

## 사용 방법
1. `index.html` 파일을 브라우저로 엽니다 (더블클릭 또는 모바일에서 열기).
2. 알바생은 검색창에 메뉴/기기 이름이나 초성을 입력해 검색합니다.
3. 사장님은 우측 상단 **⚙️ 관리** → PIN `0000` 입력 후 메뉴를 직접 등록/수정합니다.
   - 처음 사용 시 PIN을 변경하고, 메뉴 입력 후 **백업 내보내기**로 한 번 저장해두세요.

## 기술
HTML5 · CSS3 · Vanilla JavaScript (Zero-Dependency)
